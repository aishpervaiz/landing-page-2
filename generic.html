Overview

These projects are structured to provide a comprehensive learning experience in both cybersecurity and software development. It starts with a series of individual cybersecurity projects through CodePath, such as CyberChef CTFs, which introduce basic cryptography and the cybersecurity mindset, and progressively covers advanced topics like password cracking, DNS exploits, malware detection, and steganography. Not only that, I have developed a console-based Movie Database Management System that allows users to add, search, and delete movie entries using object-oriented programming principles. The system handles file I/O operations to ensure data persistence and provides an interactive interface for user input.
